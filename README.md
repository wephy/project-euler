# Project Euler Solutions [project-euler]

**Project Euler solutions. I attempt to find the perfect balance between making extremely efficient/optimised code, 'code-golfing', and also having the solution be understandable/intuitive.**

For now only problems 001-100 are posted visibily in accordance with [Project Euler](https://projecteuler.net/)'s request. Subsequent problems are encrypted with the problems' answer as the key.

## Decrypting problems
To decrypt any particular problem-file you must know the solution to the problem.

1. Run `decrpyt.py`
2. You will be prompted for the number of the problem you wish to decrypt
3. Next you will need to enter the problems' solution
4. Files in all languages should then be replaced with their correct code.
